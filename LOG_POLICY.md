# LOG_POLICY.md — LAB_PUBLIC

Source-Exempt: operational-doc

_公開前チェックリスト / 最終更新: 2026-03-07_

## 目的

このリポジトリに置く内容が、公開しても安全で再利用可能な状態かを確認する。

## スコープ

- 知見メモ・調査ログ: 出典 URL 必須
- 運用ドキュメント・テンプレート・インフラ補助資料: `Source-Exempt: operational-doc` を明示すれば URL 必須の対象外

## 必須チェック

- [ ] 知見メモ・調査ログには出典 URL がある
- [ ] 個人情報、顧客情報、秘密情報、推測を書いていない
- [ ] ローカル絶対パスを書いていない
- [ ] 固定 IP、認証情報、トークン、パスワードを直書きしていない
- [ ] 共有相手や共有日時など、運用記録が必要な場合は private 側にのみ残す

## infra ファイルの追加チェック

- [ ] `.env` に逃がすべき値を追跡ファイルへ書いていない
- [ ] イメージタグまたは digest が固定されている
- [ ] 復旧手順が repo 内のドキュメントにある

## 補足

- 公開可否に迷う内容は、LAB_PUBLIC ではなく private 側で扱う
- 自動チェックは `scripts/check-public-safety.ps1` を使う
